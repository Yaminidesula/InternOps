# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly instead of creating a public GitHub issue.

Please contact the project maintainers through GitHub or another private communication channel.

## Response Timeline

- Initial response: Within 48 hours
- Status update: Within 7 days
- Resolution timeline depends on the severity of the issue

## Scope

Please report vulnerabilities related to:

- Authentication and authorization
- Sensitive data exposure
- Dependency vulnerabilities
- Remote code execution
- Privilege escalation

Thank you for helping improve the security of InternOps.
